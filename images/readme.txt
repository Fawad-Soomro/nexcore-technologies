## üé® Image Guidelines

### Format
- **Preferred**: JPG for photos, PNG for logos/graphics
- **Compression**: Optimize for web (use tools like TinyPNG)
- **File Size**: Keep under 500KB per image when possible

### Quality
- **Resolution**: High quality but web-optimized
- **Aspect Ratio**: Maintain consistent aspect ratios
- **Style**: Professional, modern, consistent across all images

### Naming Convention
- Use lowercase
- Use hyphens instead of spaces
- Be descriptive
- Match the names exactly as specified above

### Brand Consistency
- Use consistent color grading across all images
- Ensure images align with NexCore's modern, tech-forward brand
- Avoid overly stock-looking photos when possible

## üîß How to Add Images

1. **Prepare Your Images**
